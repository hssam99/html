DOM: Document Object Model 문서의 객체를 지칭하는 용어(HTML 안에 입력된 객체)

  • html 하위에 있는 모든 태그들 (element) / 속성(attribute) = property /
  • element 가지고 올때 : getElementBy** / ID, class, tagname, querySelector 등
  • getElementBy : 단일 엘리먼트 리턴
  • byClassName / byTagName : 복수 엘리먼트 리턴
  • HTMLCollection 타입의 유사 배열로 리턴 Index 사용하여 선택
